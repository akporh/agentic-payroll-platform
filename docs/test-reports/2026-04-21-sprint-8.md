## Bugs Found and Fixed During Testing

### BUG-1 (Fixed): `unit` and `base` SelectField values violated DB constraints
- **Root cause:** `AddRateCodeSlideOver` used `'hours'`/`'days'` but `ck_rcr_unit` requires `'hour'`/`'day'`. `'basic_monthly'` not allowed by `ck_rcr_base` (only `'basic_hourly'`/`'basic_daily'`).
- **Fix:** SelectField options corrected; default state updated.
- **Discovered:** Testing via `POST /rate-codes` returning 500.

### BUG-2 (Fixed during implementation): `code` missing from salary definition creation
- **Root cause:** `SalaryDefinitionCreateSchema`, `onboarding_service.create_salary_definition`, route call, and `workspace.ts` all omitted `code` despite the DB column being `NOT NULL`.
- **Fix:** `code` added to all 4 layers.
- **Discovered:** Code review while wiring `AddSalaryDefSlideOver`.
